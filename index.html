Drinking Gun

Interdisciplanary Project

This gun was designed for a costume. The main goal was to shoot drinking liquids using air pressure and a self made valve system. This was achieved by integrating a small ball pump in the cardboard design that can be used by pumping the reload grip. To control the liquids system I designed a small magnetic valve using tubing and a small metal ball. To improve the appeal a rasperry pi pico was used to control an advanced lighting system taking effect when shooting liquids.
